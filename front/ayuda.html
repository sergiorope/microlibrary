Este es un proyecto que inicié con la idea de trabajar con una nueva arquitectura, más compleja y desacoplada, donde a dia de hoy se trabaja mucho, la parte principal es el backend el cual está separado completamente del front y el cual esta construido como ya se ha dicho a partir de una aquitectura basada en microservicios con el framework de SpringBoot, utilizando dentro de el diferentes tecnologías, también tanto el back como este front, han sido dockerizados a partir de la herramienta de Docker para una mayor accesibilidad a cualquier persona que pruebe la aplicación ademas de su correspodiente orquestación, a medida que vaya adquiriendo nuevos conocimientos se añadirán nuevas funcionalidades.
